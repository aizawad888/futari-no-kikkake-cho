# README

## サービス概要
日常の小さな変化に気付いてほしいけど、恥ずかしくて言えないことをやさしくサポートするアプリです。
投稿はカテゴリ選択で行い、相手にはヒントだけが届きます。
答えは一定期間後に公開され、リアルでの「ありがとう」を自然に伝えられる仕組みです。

## このサービスへの思い・作りたい理由
日常の小さな気付きはとても大事だけど、些細なことを伝えるのは恥ずかしいものです。
「前髪切った」「トイレットペーパー替えた」「部屋片付けた」など、気付いてほしいことは多いけれど、口に出すのは少し照れくさい。
そのため、アプリを通じてさりげなく相手に気付きを促し、自然なコミュニケーションのきっかけを作りたいと考えました。

## ターゲットユーザー

- カップルや夫婦
- お互いの日常の小さな変化に気付いてほしいけど、言いにくいと感じる人
- 記念日など大事な日を忘れずに祝いたい人
- 日々の家事や生活の細かい変化を楽しく共有したい人

## サービスの利用イメージ

| 課題 | なぜそれが困るのか？ | アプリがどう解決する？ |
| --- | --- | --- |
| 言いにくい小さな変化に気付いてほしい | 自分から言うのは恥ずかしい、でも気付いてほしい | カテゴリだけ選ぶ投稿で、相手にはヒントのみ通知 |
| 相手が気付けるか不安 | 気付けなかったら悲しい気持ちになる | 一定期間後に答えが見られる仕組みでモヤモヤを解消 |
| ありがとうを言うタイミングがわからない | 思ったことを口頭で伝えにくい | 答え公開時に固定文で「ありがとう」を伝えるヒントを提示 |
| 記念日を忘れやすい | 生活に追われて大事な日を忘れてしまう | 事前にリマインダーとメッセージ提案を表示 |

## サービスの差別化ポイント・推しポイント
- 日記やつぶやきでもタスク管理でもなく、日ごろの「ありがとう」や「いいね」といったリアルコミュニケーションのサポートとして使えるアプリ。
- 気付いてほしいこと、褒められたらうれしいこと、新しく始めたことなど「会話や感謝のきっかけ」をシンプルな仕組みで「気付かせる」を実現
    - カテゴリと内容を投稿（テンプレート化可能）
    - 相手に通知が送られる「外見に小さな変化があったかも」「家の中で変化があったかも」など、カテゴリにそった固定分のみ
    - 一定時間後、内容を相手が確認可能（一定時間は投稿側が時間を選べる）
    - 気付いてほしいレベルを設定可能（相手には応じたUIで表示される。赤文字とか一番上にとか）
- 露骨にならず、恥ずかしくても投稿しやすいカテゴリ選択形式。
    - ex. カテゴリ「家事」
        - トイレットペーパーを変えた
        - ゴミを出した
        - 洗濯をした
        - …
- 反応ゼロが可視化されないため、凹むことがない。その代わりにリアルコミュニケーションのヒントを出す。
- 一定期間の答えロックでワクワク感とモヤモヤ解消を両立
- 記念日や家事の変化など日常の小さな気付きもカバー

## 機能候補

### MVP
- カテゴリ選択+内容入力による変化投稿
- 相手へのアプリ内通知（ぼんやりしたヒントのみ）
- 一定期間後の答え公開機能
- 固定文による「ありがとう」ヒント表示

### 本リリース
- 内容のテンプレート化（プリセット化）
- 投稿のアーカイブ・検索機能

### 追加機能
- 記念日リマインダー
- 週末に投稿をまとめて振り返る機能
- プッシュ通知

## 使用する技術スタック
- 使用するフレームワーク: Ruby on Rails、JavaScript
- データベース: PostgreSQL
- デプロイ先: Render
- 使用予定のライブラリ: Devise（認証用）、Omniauth（SNS認証）、webpush gem（プッシュ通知）
